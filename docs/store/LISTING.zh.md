# Chrome Web Store 商店描述（中文）

---

## 名称（≤ 45 字符）

```
WS Inspector — WebSocket 调试工具
```

## 简短描述（≤ 132 字符）

```
实时抓包 WebSocket 帧，自动解压 gzip/deflate，JSON 高亮，深色主题，WSS 连通性诊断，发包模板，支持 DevTools 面板与独立侧栏。
```

## 类别

`开发者工具 (Developer Tools)`

---

## 详细描述

```
WS Inspector 是一款专为 Chrome 打造的全功能 WebSocket 调试扩展。
无需代理或中间人抓包，即可截获当前页面所有 WebSocket 帧（包括压缩二进制帧），
在精心设计的深色三栏面板中逐帧查看与分析。

═══════ 1.2 新功能 ═══════

✦ 深色黑客主题 UI — 三栏布局（图标导航 | 消息列表 | 详情查看器），
  单行工具栏，设计参考 Postman / Charles Proxy
✦ WSS 诊断 tab — 检测 WebSocket 端点是否可达：
  DNS 解析 → TLS → 握手，逐步时间轴展示耗时
✦ 本地发包模板 — 在发包 tab 中保存、载入、删除常用报文预设
✦ 键盘导航 — ↑ / ↓ 在消息列表中移动，← / → 切换详情 tab

═══════ 核心功能 ═══════

✓ 截获页面所有 WebSocket 连接（包括动态创建的连接）
✓ 自动解压 — gzip / deflate / deflate-raw，魔数字节自动检测
✓ 四种详情视图：解码 JSON（语法高亮）/ 原始文本 / 十六进制 / 连接信息
✓ 主动连接 — 直接从面板建立 ws:// / wss:// 连接并发包
✓ 心跳保活 + 指数退避自动重连
✓ 订阅规则 — 连接成功后自动发送帧（认证 token、订阅报文）
✓ 消息高亮 — 关键词 / 正则 / JSON 字段匹配着色
✓ 消息置顶 — 重要帧固定在列表顶部
✓ 方向过滤 — 仅显示发送 ↑ 或接收 ↓
✓ 发送模板（全局）— 常用报文一键发送
✓ 右键菜单 — 复制 / 重发 / 置顶 / 删除
✓ JSON 导出 — 将当前过滤后的消息记录导出为文件
✓ 实时消息计数 + badge

═══════ 两种显示模式 ═══════

① DevTools 面板（F12 → "WS Inspector" 标签页）
   嵌入 Chrome 开发者工具，与 Network / Console 并列显示

② 浏览器侧栏（点击工具栏图标）
   固定在页面右侧，自动跟随标签页切换，
   页面导航时自动清空，无需打开 DevTools

═══════ 适用人群 ═══════

• 前端 / 全栈开发者：调试 WebSocket API、实时数据推送、在线聊天
• 测试工程师：验证推送消息报文与服务端行为
• 协议分析人员：分析游戏客户端、行情数据流、IM 应用流量
• 任何需要在写客户端代码前快速验证 WSS 端点是否可达的人

═══════ 隐私声明 ═══════

• 100% 本地运行 — WebSocket 数据不会上传到任何服务器
• 不收集、不存储任何用户信息
• 消息仅保存在内存中，关闭面板后自动释放
• 完全开源，代码可审计

═══════ 使用方法 ═══════

1. 安装扩展
2. 打开任意使用 WebSocket 的页面（交易平台、聊天、游戏…）
3. 点击工具栏图标打开侧栏，或按 F12 → WS Inspector 标签页
4. 刷新页面 — 消息帧自动出现，按连接分组显示
5. 点击任意消息查看详情；用 ↑↓ 导航，← → 切换详情 tab
6. 点击"＋ 新建连接"建立自己的 WebSocket 并发包
7. 使用 ⚡ 诊断 tab 检测 WSS 端点是否可达
```
