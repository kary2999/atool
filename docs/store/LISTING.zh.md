# Chrome Web Store 商店描述（中文）

---

## 名称（≤ 45 字符）

```
DevToolbox — WebSocket 抓包 + 开发工具箱
```

## 简短描述（≤ 132 字符）

```
抓包 WebSocket 的同时，内置多语言 SQL 生成、二维码、JSON 格式化、时间戳换算、文本对比等开发工具。深色面板，支持 DevTools 面板与独立侧栏。
```

## 类别

`开发者工具 (Developer Tools)`

---

## 详细描述

```
DevToolbox 是一款面向开发者的 Chrome 工具箱扩展。
它以一款成熟的 WebSocket 抓包/调试器为核心，
并把日常高频用到的开发小工具收纳进同一个深色三栏面板，
切换即用，无需在多个网页工具之间来回跳转。

═══════ 2.0 新增：开发工具组 ═══════

⛁ SQL 生成 — 输入中文文案，自动翻译为 18 种语言并生成
   static_lang / static_lang_error 的 INSERT SQL，带语法高亮，一键复制
⌕ SQL 检索 — 粘贴已有 INSERT 语句，按 source / error_tag / 任意文案
   实时检索过滤，命中高亮，可查看单条字段明细并复制
▦ 二维码生成 — 文本 / 链接转二维码，支持中文，纠错级别与像素可调，
   导出 PNG
❴❵ JSON 工具 — 格式化 / 压缩 / 去注释 / 排序键 / 转义 / 去转义，带合法性校验
◷ 时间日期 — 时间戳 ↔ 日期互转，多时区换算（按所选时区解释输入），
   秒 / 毫秒 / ISO / 星期一并展示
⇄ 文本对比 — 两段文本逐行 diff，新增绿、删除红，带增删行数统计

═══════ WebSocket 抓包核心 ═══════

✓ 截获页面所有 WebSocket 连接（包括动态创建的连接）
✓ 自动解压 — gzip / deflate / deflate-raw，魔数字节自动检测
✓ 四种详情视图：解码 JSON（语法高亮）/ 原始文本 / 十六进制 / 连接信息
✓ 主动连接 — 直接从面板建立 ws:// / wss:// 连接并发包
✓ 心跳保活 + 指数退避自动重连
✓ WSS 诊断 — DNS → TLS → 握手逐步时间轴，检测端点是否可达
✓ 订阅规则 — 连接成功后自动发送帧（认证 token、订阅报文）
✓ 消息高亮（关键词 / 正则 / JSON 字段）、置顶、方向过滤
✓ 发送模板、右键菜单（复制 / 重发 / 置顶 / 删除）、JSON 导出

═══════ 两种显示模式 ═══════

① DevTools 面板（F12 → "DevToolbox" 标签页）
   嵌入 Chrome 开发者工具，与 Network / Console 并列

② 浏览器侧栏（点击工具栏图标）
   固定在页面右侧，自动跟随标签页切换，无需打开 DevTools

═══════ 适用人群 ═══════

• 前端 / 全栈开发者：调试 WebSocket、实时推送、在线聊天
• 后端 / 测试工程师：生成与检索多语言文案 SQL、验证推送报文
• 协议分析人员：分析行情数据流、IM、游戏客户端流量
• 任何需要二维码、JSON、时间戳、文本对比等高频小工具的人

═══════ 隐私声明 ═══════

• WebSocket 抓包 / 解码 / 诊断 / 发包 —— 100% 本地运行，数据不出本机
• 二维码 / JSON / 时间日期 / 文本对比 —— 全部本地计算，不联网
• ⚠ 唯一例外：「SQL 生成」的多语言翻译会把你输入的中文文案发送到
  Google 翻译接口（translate.googleapis.com）以获取译文；
  不使用该功能时，扩展不会产生任何外发网络请求
• 不收集、不存储任何用户身份信息；抓包消息仅保存在内存，关闭即释放
• 完全开源，代码可审计

═══════ 使用方法 ═══════

1. 安装扩展
2. 点击工具栏图标打开侧栏，或按 F12 → DevToolbox 标签页
3. 抓包：打开任意使用 WebSocket 的页面并刷新，消息按连接分组自动出现
4. 工具：在左侧导航「工具」组切换 SQL / 二维码 / JSON / 时间 / 对比
5. 各工具左侧输入、右侧实时出结果，支持一键复制 / 导出
```
