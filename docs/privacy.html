Privacy Policy

Last updated: 2026-05-20  ·  Applies to version 1.2.0 and later

🔒

Short version: Atool collects absolutely no data. Everything stays in your browser. There is no backend, no analytics, and no telemetry of any kind.

1. Developer

Atool is an independent open-source project maintained by a solo developer.

Contact: kary372022@gmail.com

2. Data We Collect

None.

  • Atool intercepts and displays WebSocket messages locally inside your browser.
  • All message data exists only in browser memory for the lifetime of the panel.
  • There is no backend server receiving any data from the extension.
  • There is no analytics, crash reporting, or telemetry.

3. Data Stored Locally

DataPurposeWhere
WebSocket URLs, frames, direction Displayed in the panel in real time Browser memory only — cleared when panel closes
Subscription rules, send templates Restored when you reopen the panel Browser localStorage (your device only)

4. When Data Leaves the Browser

Only when you explicitly click "Export JSON" — this writes the current message log to your local Downloads folder. No data is sent to any server at any point.

When you use the "New Connection" feature, the extension establishes a WebSocket connection to an address you type. That outbound connection is your own request to your own target; the extension does not route or log it externally.

5. Third-Party Requests

Atool makes no third-party network requests of its own — no CDNs, no fonts, no tracking pixels, no error-reporting services.

6. Permissions Explained

PermissionReason
storageSave subscription rules and send templates to localStorage
sidePanelShow the debugger panel in the browser's side panel
tabsFollow the active tab in Side Panel mode
Content scripts on <all_urls>Users debug WebSocket on arbitrary sites; the domain can't be predicted in advance

Permissions not used: webRequest, cookies, history, identity, remote code execution.

7. Children

Atool is a developer tool not directed at children. It does not knowingly collect data from anyone.

8. Open Source

The complete source code is available at github.com/kary2999/atool. Anyone can audit exactly what the extension does.

9. Changes to This Policy

If future versions require any new data handling, this document will be updated and the change will be noted in the extension's release notes.

10. Contact

Privacy questions or vulnerability reports: kary372022@gmail.com
